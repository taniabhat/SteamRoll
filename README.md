## ✨ Features

- **AutoPipeline** — Build end-to-end data pipelines with drag-and-drop logic. SteamRoll generates the underlying code automatically, reducing boilerplate by 90%.
- **Real-Time Analytics** — Sub-200ms query latency on billion-row datasets with live dashboards that react instantly to streaming data.
- **Smart Routing** — AI-powered decision routing directs each data event to the right transformation, model, or destination automatically.
- **Deep Insights** — Multi-dimensional pattern recognition surfaces anomalies, correlations, and predictions.
- **Enterprise Security** — SOC 2 Type II, GDPR-compliant data handling with field-level encryption, RBAC, and full audit trails.

## 🖥️ Tech Stack

| Layer        | Technology                                              |
| ------------ | ------------------------------------------------------- |
| Structure    | HTML5 (semantic, accessible)                            |
| Styling      | Vanilla CSS with custom properties & claymorphism       |
| 3D Visuals   | [Three.js](https://threejs.org/) (r128)                 |
| Typography   | [Inter](https://fonts.google.com/specimen/Inter) + [JetBrains Mono](https://fonts.google.com/specimen/JetBrains+Mono) via Google Fonts |
| Animations   | CSS keyframes + JS IntersectionObserver scroll reveals   |

---

## 🎨 Design Highlights

- **Claymorphism UI** — Soft, elevated card elements with frosted-glass backdrop blur and layered box shadows.
- **Warm Gold & Oceanic Dark palette** — Forsythia (`#FFC801`), Deep Saffron (`#FF9932`), Nocturnal (`#114C5A`), Oceanic Noir (`#172B36`).
- **3D Hero Section** — Interactive particle field with floating torus rings and a rotating orb, all rendered via Three.js.
- **Responsive Layout** — Desktop bento grid transforms into an accordion on mobile with animated context transfer.
- **Multi-currency Pricing** — Dynamic pricing matrix supporting USD, INR, and EUR with monthly/annual toggle and animated transitions.
- **Accessibility** — ARIA labels, keyboard navigation, `prefers-reduced-motion` support, and semantic HTML throughout.

## 🚀 Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/taniabhat/SteamRoll.git
   cd SteamRoll
   ```

2. **Open in browser**
   ```bash
   # Simply open the HTML file
   start "index (1).html"       # Windows
   open "index (1).html"        # macOS
   xdg-open "index (1).html"   # Linux
   ```

   Or use a local dev server:
   ```bash
   npx serve .
   ```
